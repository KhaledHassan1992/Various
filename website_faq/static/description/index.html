Website FAQ

An Odoo FAQ module allows you to post all Frequently Asked Questions in the back-end and get them in a FAQ page.
This allows to reduce the amount of inbound emails and phonecalls to your customer support. You can insert also snippets inside the body of the answer.

Very easy to post a question

Just go the website setting you find a menu FAQ just add a FAQ.


You will get then a menu FAQ and a bootstrap accordion with enumerated questions and answers.

How to install it

  • Copy dependent module inside the folder "Modules" and put it inside the addons folder of your odoo
  • Delete "Modules" folder from website_faq folder .
  • Put also the folder website_faq inside the addons folder.
  • Open __openerp__.py and uncomment dependency modules.
  • Search for "website_faq" inside your odoo back-end and install it
